Scrape assignments from Viggo effortlessly.

Viggoscrape is an API for getting data from Viggo. With Viggoscrape, you are one simple web request away from getting relevant assignments without ever touching Viggo.

Looking to use Viggoscrape?

Here's what this tool can offer.

Simplicity

With API wrappers for Python and Node.js, getting Viggo assignments in JSON form has never been simpler.

Possibility

Need data from half a year ago? No problem. Need the data categorized by data type instead? That's done.

Affordability

Viggoscrape is completely free, open source software. And it will always be that way.

Flexibility

Because Viggoscrape is a web API, you can use it anywhere.
