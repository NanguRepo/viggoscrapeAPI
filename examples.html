available is great and all, but how do you actually integrate these
tools into your projects? Let's have a look at that.

Implementing Viggoscrape

@nangu March 16, 2022

Interacting with the Viggoscrape API using simple web requests is, of course, an option, but there are a few, more fleshed out, API wrappers available.

These options can be used in various applications, for example as an embed or to create a notifier.

So far I've only made 2 API wrappers for Viggoscrape, but more will come soon. The wrappers are for Javascript and Python. Both have their source code hosted on GitHub, with the packages distributed on their respective package managers:

  • Viggoscrape.py
  • Viggoscrape.js

Other methods

With other languages, you'll have to make use of HTTP requests to manually interact directly with the API. This is as simple as a GET request, no POST needed. The URI syntax is found within the documentation.
